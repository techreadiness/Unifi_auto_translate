* Added `kaia_connectAndSign` function

**Request**

```
[data]
```

* `data` : data to sign

**Response**

```
[address, signature]
```

* `address` : address which is signed to data
* `signature`

**Error**

```
REQUEST_REJECTED: -32002
```

When user did cancel connection.



_**To request**_**&#x20;`kaia_connectAndSign`**_**, an SDK version upgrade is mandatory.**_

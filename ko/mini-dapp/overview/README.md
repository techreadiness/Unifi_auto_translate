# 개요

## 주요 기능

미니 앱은 LINE 메신저와 카이아 블록체인을 기반으로 개발된 웹 애플리케이션입니다. LINE의 사용자 풀을 활용하여 전 세계 약 2억 명의 사용자에게 다양한 경험과 보상을 제공합니다. LINE 미니 앱과 달리 미니 앱은 포인트나 현금형 보상뿐만 아니라 웹 3 기반 보상도 제공할 수 있습니다. 또한 미니 앱은 LINE 비사용자에게도 동일한 서비스를 제공할 수 있습니다. LINE 메신저의 공식 계정을 활용한 프로모션 및 친구 초대 기능을 통해 더 많은 LINE 기반 사용자를 확보할 수 있습니다.

또한 미니 앱은 플레이, 지갑, 결제, 보상 등 다양한 기능을 제공해야 합니다. 주요 기능은 아래와 같습니다.

미니 앱은 특정 프레임워크를 활용한 웹 애플리케이션을 구축해야 합니다. LINE 메신저를 통해 접속하는 사용자를 위한 LIFF 버전과, 비(非) LINE 메신저를 통해 접속하는 사용자를 위한 웹 버전이 필요합니다.

* **미니 앱은 더 넓은 지역을 커버하기 위해 두 가지 버전으로 구축되어야 합니다.**
  * 미니 앱을 LIFF 버전과 웹 버전으로 모두 개발해야 하는 이유는 다음과 같습니다: 특정 국가에서는 LINE 메신저 사용이 어렵거나 불가능할 수 있습니다. 또한 Web3 사용자는 LINE 버전을 사용하지 않으려 할 수 있습니다. \
    따라서 해당 국가 및 Web3 사용자를 위해 웹 버전을 별도로 개발해 주시기 바랍니다.
  * 미니 앱 - LIFF는 LINE 메신저 내에서 애플리케이션을 실행하는 버전입니다\
    (LINE 사용자용).
  * 미니 앱 - 웹은 웹 브라우저에서 애플리케이션을 실행하는 버전입니다\
    (비-LINE 사용자용).

#### 개선 사항

* OA 프로모션 2단계 지원
  * 미니 Dapp 생태계 내 사용자 유동성 증진을 위해 Dapp Portal OA 팝업 이벤트 2단계를 진행합니다.
    본 팝업은 USDT 보상을 제공하며 다양한 미니 Dapp 간 교차 프로모션을 장려합니다.
  * **프로모션 참여를 원하는 파트너사는 @dappportal_official로 문의 바랍니다.**
* 미니앱 연결 시 스크롤 시 화면이 비어 보이는 현상 수정.
* 모든 결제 API 프로세스를 **취소 불가 작업**으로 업데이트하여 보다 안정적이고 신뢰할 수 있는 실행 보장.

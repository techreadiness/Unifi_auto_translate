#### 개선 사항

* Dapp Portal 리브랜딩 지원을 위해 이번 버전이 출시되었습니다. (리브랜딩 관련 세부 사항은 별도 채널을 통해 곧 공지될 예정입니다.)
* 추가된 메서드:
  * **USDT 잔액** 조회 메서드
    * `getErc20TokenBalanceWithDepositedBalance()` 
  * **USDT 전송**을 위한 스마트 계약 서명
    * 스왑을 포함한 스마트 계약을 통해 구현된 USDT 전송의 경우, 개발자는 `kaia_sendTransaction` 요청에 `depositTokenAddress` 및 `depositAmount`를 포함해야 합니다.
    *   walletType이 WEB 또는 LIFF인 경우 변경이 필요합니다. walletType이 WEB 또는 LIFF가 아닌 경우, 개발자는 기존 매개변수를 그대로 사용해야 합니다.

        * WalletType이 WEB 또는 LIFF가 아닌 경우 (기존 그대로)

        ```
        params = {
        type: &quot;SmartContractExecution&quot;,
        from: &quot;0x4b53..dbe6&quot;,
        to: &quot;contractAddress&quot;,
        input: &quot;0xabcd...&quot;,
        }
        ```

        * WalletType이 **WEB** 또는 **LIFF**인 경우

        ```
        params = {
        type: &quot;SmartContractExecution&quot;,
        from: &quot;0x4b53..dbe6&quot;,
        to: &quot;contractAddress&quot;,
        input: &quot;0xabcd...&quot;,
        depositTokenAddress: &quot;0xd077...&quot;, // USDT 계약 주소
        depositAmount: &quot;100&quot; // 전송할 USDT 금액
        }
        ```

#### 참고

Mini Dapp SDK v1.5.0으로 즉시 업데이트할 필요는 없습니다. 공식 리브랜딩 발표 시점에 맞춰 추후 업데이트를 선택할 수 있습니다. _**추가된 메서드는 리브랜딩된 Dapp Portal 출시 후 정상 작동합니다.**_

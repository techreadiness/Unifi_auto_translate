## 참고사항

Mini Dapp SDK v1.3.0은 walletType 중 하나로 Bitget Wallet을 제공합니다.

getWalletType()의 응답은 `WalletType.BITGET`입니다.

npm: [https://www.npmjs.com/package/@linenext/dapp-portal-sdk/v/1.3.0](https://www.npmjs.com/package/@linenext/dapp-portal-sdk/v/1.3.0)

cdn: [https://static.kaiawallet.io/js/dapp-portal-sdk-1.3.0.js](https://static.kaiawallet.io/js/dapp-portal-sdk-1.3.0.js)

* 미니 Dapp SDK를 통해 Bitget Wallet을 제공하려면 SDK 업그레이드가 필요합니다.
* 코드 변경은 필요하지 않습니다.
* SDK 버전 업그레이드 전 [이 가이드](https://docs.dappportal.io/mini-dapp/mini-dapp-sdk/wallet-provider/domain-verification-via-reown)에 따라 도메인 인증을 완료하고, 기술 지원 채널을 통해 ProjectId를 제출해 주십시오.

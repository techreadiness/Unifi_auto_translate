#### 변경 사항

* 결제 상태가 CONFIRMED일 때 Webhook의 cryptoPaymentInfo 필드가 추가되었습니다

```
{
    &quot;paymentId&quot;: &quot;&quot;,
    &quot;status&quot;: &quot;CONFIRMED&quot;,
    &quot;cryptoPaymentInfo&quot;: { // status: CONFIRMED 시 추가
        &quot;paymentTxHash&quot;: &quot;0x..&quot;
    }
}  
```

* 대체 가능 토큰 잔액 조회 메서드 추가

```
walletProvider.getErc20TokenBalance(contractAddress, walletAddress)
```

* **USDT**가 `pgType: CRYPTO` 시 currencyCode로 추가됨
  * <mark style="color:blue;">**USDT 결제 시 가스 수수료 위임 지원**</mark>

Bitget Wallet 사용을 위한 도메인 인증을 아직 완료하지 않으셨다면 [이 가이드](https://docs.dappportal.io/mini-dapp/mini-dapp-sdk/wallet-provider/domain-verification-via-reown)를 참조해 주세요.

**보안 및 호환성을 보장하기 위해 항상 최신 버전의 Mini Dapp SDK 사용을 권장합니다.**

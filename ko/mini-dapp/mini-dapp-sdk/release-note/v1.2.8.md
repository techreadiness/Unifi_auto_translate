* `kaia_connectAndSign` 함수 추가

**요청**

```
[data]
```

* `data` : 서명할 데이터

**응답**

```
[address, signature]
```

* `address` : 데이터에 서명된 주소
* `signature`

**오류**

```
REQUEST_REJECTED: -32002
```

사용자가 연결을 취소했을 때 발생합니다.



_**`kaia_connectAndSign`**_**를 요청하려면 SDK 버전 업그레이드가 필수입니다.**_

# v1.3.3

## 출시일

2025년 6월 10일 07:20 (UTC)

## 참고

Mini Dapp SDK v1.3.0 이상에서 발생할 수 있는 버그가 수정되었습니다.

npm: [https://www.npmjs.com/package/@linenext/dapp-portal-sdk/v/1.3.3](https://www.npmjs.com/package/@linenext/dapp-portal-sdk/v/1.3.3)

cdn: [https://static.kaiawallet.io/js/dapp-portal-sdk-1.3.3.js](https://static.kaiawallet.io/js/dapp-portal-sdk-1.3.3.js)

\[문제 없음] SDK 초기화와 메서드 호출 사이에 충분한 간격이 있을 때

```
// index.tsx
globalSdk = await DappPortalSDK.init()

// login.tsx
const provider = global.getWalletProvider();
const accounts = provider.request({method: &#x27;kaia_requestAccounts&#x27;})
```

**\[문제]** 메서드가 제대로 실행되지 않습니다. (SDK 초기화 직후 호출 시 발생)

```
const sdk = await DappPortalSdk.init();
const provider = sdk.getWalletProvider();
const accounts = provider.request({method: &#x27;kaia_requestAccounts&#x27;})
```

위와 같은 문제가 발생할 경우, Mini Dapp SDK 버전을 v1.3.3으로 업데이트해 주십시오.



**보안 및 호환성을 보장하기 위해 항상 최신 버전의 Mini Dapp SDK 사용을 권장합니다.**

# SDK 인증 획득 방법

미니 Dapp SDK를 통합하려면 Dapp Portal 팀으로부터 `clientId`와 `clientSecret`을 받아야 합니다.

#### 1. Dapp Portal 팀과 계약 체결

#### 2. 미니 앱 정보를 포함한 SDK 이용 약관 동의서 제출

#### 3. 2단계에서 제출한 이메일로 `clientId` 및 `clientSecret` 수령

`clientSecret`은 매우 중요한 정보이므로 외부 유출을 방지하기 위해 각별히 주의하십시오.
유출이 의심될 경우 Dapp Portal 팀과 상담하여 키를 갱신하는 것이 좋습니다.

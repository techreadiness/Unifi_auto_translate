# 미니 앱

## 미니 앱 구축 방법

<figure><img src="../.gitbook/assets/스크린샷 2025-07-17 오후 2.28.26.png" alt=""><figcaption></figcaption></figure>위 다이어그램에서 미니 앱 구현에 필요한 개발 범위를 확인할 수 있습니다.

LIFF 버전의 미니 앱은 반드시 LINE Developers를 통해 구현해야 합니다. 자세한 내용은 LINE 통합 메뉴에서 확인할 수 있습니다.

미니 앱 SDK는 지갑 연동 및 결제 기능을 지원합니다. 연결되는 지갑은 LIFF 버전과 웹 버전에 따라 다를 수 있습니다. 미니 앱은 결제 기능을 통해 인앱 아이템을 판매할 수 있습니다.

또한 별도 트랙으로 정보를 등록하면 Dapp 포털에 노출될 수 있으며 NFT 판매도 가능합니다.

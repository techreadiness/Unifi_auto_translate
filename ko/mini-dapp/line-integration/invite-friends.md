# 친구 초대

## 가이드

#### 친구 초대 기능은 미니 앱(LIFF)에서만 이용 가능합니다.

#### 미니 앱(LIFF)에 사용자를 유치하고 활성화하는 데 가장 중요한 기능입니다. 다양한 추천을 통해 LINE 사용자들 사이에서 바이럴 효과를 내는 것이 중요합니다.

* 친구 초대 기능을 사용하려면 아래 LIFF API URL을 참조하세요.
  * [메시지 전송](https://developers.line.biz/en/docs/liff/developing-liff-apps/#sending-messages)
  * [ShareTargetPicket](https://developers.line.biz/en/docs/liff/developing-liff-apps/#share-target-picker)
    * 메시지가 다른 OA 계정이나 채팅 그룹이 아닌 사용자의 친구에게만 전송되도록 설정하려면, [options.isMultiple을 false로 설정](https://developers.line.biz/en/reference/liff/#share-target-picker-arguments)하세요.

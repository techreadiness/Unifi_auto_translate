# 함께하세요

## ✅ 유니파이 앱 온보딩 가이드

유니파이 앱은 다양한 버전 조합으로 온보딩할 수 있습니다.
개발자는 서비스 아키텍처와 대상 사용자에게 가장 적합한 옵션을 선택할 수 있습니다.

**사용 가능한 버전 조합**

* LINE 미니앱 &amp; LINE 로그인 LIFF &amp; 웹
* LINE 미니앱 &amp; 웹
* LINE 로그인 LIFF &amp; 웹
* LINE 미니앱 또는 웹 (단일 유형)
  * 📌 LINE 로그인 LIFF는 기술적으로 웹 서비스 상에 구현되므로 단일 유형으로 선택할 수 없습니다.\
    따라서 LINE 로그인 LIFF를 사용하는 서비스는 반드시 웹 버전과 조합하여 출시해야 합니다.

**지원되는 온보딩 유형**

| 버전         | 운영 환경 및 지원 기능                                                 | 대상 사용자   | 대상 지역 | 온보딩 요구사항                                                                                                                 | SDK 기능 차이                           | 데모 제출 권한 |
| --------------- | ---------------------------------------------------------------------------------- | -------------- | -------------- | --------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- | ------------------------- |
| LINE MINI 앱   | LINE 모바일 앱 환경 내에서 실행되며 Unifi 기능 배치 지원 | LINE 사용자     | 일본          | 일본 법인 번호(호진방고)를 보유한 법인 또는 일본 거주 개인 사업자에게만 적용됩니다. | **IAP 결제만 지원**                    | LY &amp; LINE NEXT            |
| LINE 로그인 LIFF | LINE 모바일 앱 환경 내에서 운영되며 유니피(Unifi) 추천 광고 위치 지원 | LINE 사용자     | 글로벌         | -                                                                                                                                       | **암호화폐 및 스트라이프(법정화폐) 결제만 지원** | LINE NEXT                 |

## 1. 유니피 앱 신청서 제출

온보딩을 시작하려면 지정된 **유니피 앱 신청서**를 제출해 주십시오.\
제공된 정보는 서비스 자격 및 온보딩 준비 상태 평가에 활용됩니다.

[Unifi Apps 신청하기 &gt;](https://tally.so/r/D4BNjp)

## 2. 내부 검토

당사 팀이 제출된 신청서를 검토하여 Unifi Apps 온보딩 자격 여부를 결정합니다.

* 내부 검토는 주로 LINE 미니앱 출시 자격, 핵심 게임 메커니즘, 신청서에 제출된 자료에 중점을 둡니다.
* 이 단계에서 초기 협력을 지원하기 위해 전담 BD(비즈니스 개발) 매니저가 배정될 수 있습니다.

## 3. 온보딩 절차 시작

* 서비스가 내부 검토를 통과하면, 실사 양식 및 Dapp 정보 등록 양식을 포함한 기타 필수 온보딩 자료와 함께 Unifi Apps SDK 이용약관 양식을 제공해 드립니다.
* 안내에 따라 유니파이 앱 SDK 이용약관을 검토 및 제출해 주십시오.
* 승인 시 유니파이 앱 SDK와 모든 필수 인증 자격 증명이 제공됩니다.
* SDK는 등록된 이메일 주소로 영업일 기준 약 3일 이내에 전달됩니다.

## 4. 유니파이 앱 데모 개발

결제 연동 방식은 선택한 온보딩 버전에 따라 다릅니다.

| 버전         | 지갑 통합 방식 | 지원 결제 수단       |
| --------------- | ------------------ | ------------------------------- |
| LINE 미니 앱   | 지원됨          | IAP 결제 (_출시 예정_)    |
| LINE 로그인 LIFF | 지원됨          | 암호화폐 및 Stripe(법정화폐) 결제 |
| 웹             | 지원됨          | 암호화폐 및 Stripe(법정화폐) 결제 |

제공된 개발 [체크리스트](https://docs.dappportal.io/~/revisions/PsJIlYlQPCCZu69maZ4H/unifi-apps-review-guidelines)에 따라 유니파이 앱을 개발해 주세요.
일반적인 문제 및 해결 방법에 대해서는 [FAQ](https://docs.dappportal.io/~/revisions/PsJIlYlQPCCZu69maZ4H/join-us-1/faq)를 참조하십시오.
개발 중 기술적 문의 사항이 있을 경우 다음 채널을 통해 언제든지 연락 주시기 바랍니다:

* [텔레그램 기술 지원 채널](../join-us/contact-us.md)
* 이메일: **dapp-support@linecorp.com**

## 5. 검토 및 피드백

### 검토 권한 및 일정

{% hint style=&quot;info&quot; %}
검토 과정은 서비스 적격성, 기능 완성도 및 규정 준수 요건에 중점을 둡니다.\
**LINE 미니앱의 경우, LY 제출 전 LINE NEXT의 내부 검토가 필수입니다.**
{% endhint %}

| 버전         | 데모 제출/검토 권한                                                                                                                               | 검토 일정        |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------------------- |
| LINE 미니앱   | <ul><li>LINE NEXT (사전 검토)</li><li>LY (최종 승인)</li><li><strong>LINE NEXT 검토 완료 후</strong> LY 제출 필요</li></ul> | 5–10 영업일     |
| LINE 로그인 LIFF | <ul><li>LINE NEXT</li><li>이메일 제출</li></ul>                                                                                                       | 영업일 기준 최대 3일 |
| 웹             | <ul><li>LINE NEXT</li><li>이메일 제출</li></ul>                                                                                                       | 영업일 기준 최대 3일  |

📌 _LINE MINI 앱의 경우, 사전 LINE NEXT 검토 없이 LY 제출은 지원되지 않습니다._

### 데모 제출

**LINE MINI 앱**

* **검토 대상**: 스테이징 버전
* **검토 절차:**
  1. 스테이징 빌드를 **사전 검토를 위해 LINE NEXT에 제출**
     1. **LINE Developers Console**을 통해 LINE MINI App 채널에 **LINE NEXT 검토 관리자**를 **관리자로 추가**해 주세요.
     2. 제출 시 **담당 검토자의 연락처 정보**가 공유됩니다.
  2. LINE NEXT로부터 확인을 받은 후,
     **LINE Developers Console을 통해 LY 승인을 요청**하세요
* **검토 피드백:**\
  검토 단계에 따라 **LINE NEXT 및 LY**를 통해 피드백이 공유됩니다.

📌 _LY 제출 전 LINE NEXT 사전 검토는 필수입니다._

**LINE 로그인 LIFF &amp; 웹**

* **검토 대상**: LINE 로그인 LIFF / 웹 버전
* **제출 방법**: **`dapp-support@linecorp.com`**로 이메일 제출
* 요청 이메일에는 다음 정보가 반드시 포함되어야 합니다:
  * 유니파이 앱(Unifi Apps) 이름
  * 데모 URL
  * 희망 출시일

### 데모 승인 후

데모가 승인되면 정식 출시를 위한 온보딩 프로세스가 시작됩니다.

* 출시 준비 절차에 대한 안내가 제공됩니다
* 운영 및 기술 점검이 수행됩니다

* 출시 전 최종 기술 및 운영 검증이 완료됩니다

# Review Guidelines

The Kaia Wave team reviews the Mini Dapp demo version based on the criteria below.

### Self-Checklist <a href="#self-checklist" id="self-checklist"></a>

|                            |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| -------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Platform Support           | <ul><li>Are both LIFF(LINE) and Web versions supported?</li></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| SDK                        | <ul><li>Is the <a href="https://docs.dappportal.io/mini-dapp/mini-dapp-sdk/release-note">latest version of the Mini Dapp SDK</a> applied?</li><li>Has the <a href="https://docs.dappportal.io/mini-dapp/mini-dapp-sdk/wallet-provider/domain-verification-via-reown">ProjectId</a> been generated via Reown and the domain successfully verified?</li></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| Wallet Connect Flow        | <ul><li>Does the <a href="https://docs.dappportal.io/lines-mini-dapp-and-dapp-portal/how-to-build-successful-mini-dapp#line-version"><strong>LINE version</strong></a> follow the Wallet Connect flow below? <code>Access Mini Dapp (LIFF) → Consent to Channel → Add Official Account → Launch Dapp → Wallet Connect (e.g. at payment or reward step)</code></li><li>Does the <a href="https://docs.dappportal.io/lines-mini-dapp-and-dapp-portal/how-to-build-successful-mini-dapp#web-version"><strong>Web version</strong></a> follow the Wallet Connect flow below? <code>Access Mini Dapp (Web) → Wallet Connect → Launch Dapp</code></li><li>Is the <a href="https://docs.dappportal.io/lines-mini-dapp-and-dapp-portal/how-to-build-successful-mini-dapp#provision-of-information-about-connected-wallet">connected wallet address</a> clearly shown to the user?</li><li>Is the <code>disconnectWallet</code> feature available?</li></ul>               |
| Payment Features           | <ul><li>Is <a href="https://docs.dappportal.io/lines-mini-dapp-and-dapp-portal/how-to-build-successful-mini-dapp#provision-of-in-app-item-store">in-app item</a> payment supported via the SDK? (Both Crypto and Stripe must be supported)</li><li>Are <a href="https://docs.dappportal.io/mini-dapp/mini-dapp-sdk/payment/policy/refund#text-that-should-be-displayed">purchase precautions</a> shown before payment?</li><li>Are <a href="https://docs.dappportal.io/mini-dapp/mini-dapp-sdk/payment/policy/payment#provide-clear-notifications">payment status notifications</a> (UI/UX) properly provided?</li><li>Is the <code>openPaymentHistory()</code> feature available?</li><li>Are fiat/crypto prices displayed based on real-time rates? (e.g., CMC, Kaia Open API)</li></ul>                                                                                                                                                                        |
| LINE Integration           | <ul><li>Are LINE Login, Messaging API channels, and a <strong>Published LIFF</strong> created via <a href="https://docs.dappportal.io/mini-dapp/line-integration/line-developers">LINE Developers</a>?</li><li>Is the Official Account (OA) properly linked and <code>aggressive</code> set? <code>Go to LINE Developers > LINE Login Channel > LIFF > Add friend option, and set it to On (aggressive)</code></li><li>Is the OA Rich Menu configured according to the <a href="https://docs.dappportal.io/mini-dapp/line-integration/official-account#template">design guide</a>?</li></ul>                                                                                                                                                                                                                                                                                                                                                                      |
| Invite Friends             | <ul><li>In the <a href="https://docs.dappportal.io/mini-dapp/line-integration/invite-friends"><strong>LIFF version</strong></a>, is <code>ShareTargetPicker</code> implemented for inviting friends?</li><li>In the <strong>Web version</strong>, is a “copy invite link” feature available?</li></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| UX/UI                      | <ul><li>Is the<a href="https://docs.dappportal.io/lines-mini-dapp-and-dapp-portal/how-to-build-successful-mini-dapp#provision-of-multi-language"> language localization</a> implemeted based on user's browser settings or IP? (English and Japanese must be supported)</li><li>Is the <a href="https://docs.dappportal.io/mini-dapp/design-guide#browser-tab-name">browser tab title</a> in the format <code>{Mini Dapp Name} | Mini Dapp</code>?</li><li>Is <a href="https://docs.dappportal.io/mini-dapp/design-guide#opengraph">OpenGraph</a> properly set for the Mini Dapp’s URL?</li><li>Does the Connect button comply with the <a href="https://docs.dappportal.io/mini-dapp/design-guide#connect-button">Dapp Portal design guideline</a>?</li><li>Is a <a href="https://docs.dappportal.io/lines-mini-dapp-and-dapp-portal/how-to-build-successful-mini-dapp#provision-of-close-confirmation-dialog">Close Confirmation Dialog</a> provided?</li></ul> |
| Security                   | <ul><li>Never expose wallet private keys, clientSecret, or other sensitive credentials in frontend code, version control, or unsecured environments. These credentials must be securely stored on the backend or in a protected secrets management system.</li></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| Game Content Determination | <ul><li><p>Please review the <em>Game Content Determination Checklist</em>.</p><ul><li>Is the primary purpose of the Mini Dapp to provide entertainment or fun?</li><li>Do different outcomes occur based on user actions?</li><li>Are there clear rules and constraints?</li><li>Is there a game-like flow with start, progress, and end?</li><li>Are there obstacles to overcome and a sense of accomplishment?</li></ul></li><li>If the Mini Dapp is determined to qualify as a game, make sure to refer to the <a href="https://docs.dappportal.io/dapp-portal/policy/grac-south-korea">GRAC guideline document</a>.</li></ul>                                                                                                                                                                                                                                                                                                                                |

### Compliance <a href="#compliance" id="compliance"></a>

|                                     |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Gambling-related                    | <ul><li>Specifications that provide cash or virtual assets as rewards based on probability using items purchased through payment are not permitted.</li><li>Specifications that provide gacha mechanisms that can offer substantial value based on probability using in-app currency or virtual assets are not permitted.</li><li>Specifications that allow participants to receive cash rewards through prize lotteries or slot games by paying a fee are not permitted.</li></ul> |
| Violence                            | <ul><li>Games or content that contain extreme violent elements are not permitted.</li></ul>                                                                                                                                                                                                                                                                                                                                                                                         |
| Crime and Drug                      | <ul><li>Content that promotes crime, drug use, abusive behavior, or inappropriately depicts alcohol and tobacco use is not permitted.</li></ul>                                                                                                                                                                                                                                                                                                                                     |
| Discrimination and Hate             | <ul><li>Content that contains derogatory remarks or statements regarding race, gender, or sexual orientation is not permitted.</li></ul>                                                                                                                                                                                                                                                                                                                                            |
| Explicitness                        | <ul><li>Any content that is sexually explicit or contains sexual depictions is not permitted.</li></ul>                                                                                                                                                                                                                                                                                                                                                                             |
| Securities                          | <ul><li>Services that have securities are not permitted (where there is an expectation of profit, investment of assets, the invested assets are in a joint enterprise, and profits come from a third party).</li></ul>                                                                                                                                                                                                                                                              |
| Intellectual Property               | <ul><li>Services that use intellectual property that is not owned for commercial purposes are not permitted.</li></ul>                                                                                                                                                                                                                                                                                                                                                              |
| Prize Contest Law                   | <ul><li><p>In the case of rewards that are contingent upon payment, the following criteria must be met.</p><ul><li>Paid below 1,000 YEN ; Max reward 200 YEN</li><li>Paid equal to/over 1,000 YEN ; Max reward 20% of paid volume</li></ul></li></ul>                                                                                                                                                                                                                               |
| Comprehensive regulatory compliance | <ul><li>Additionally, services that are generally difficult to reference based on societal norms may not be permitted depending on the review by the Dapp Portal team.</li></ul>                                                                                                                                                                                                                                                                                                    |

# NFT 발행 및 드롭 설정 방법

{% stepper %}
{% step %}
## NFT 컬렉션 및 드롭 정보 제출

NFT<a href="#id-1.-submit-information-for-nft-collections-and-drops" id="id-1.-submit-information-for-nft-collections-and-drops"></a>

드롭을 출시하거나 NFT 에어드롭을 진행할 계획이라면, 해당 NFT 컬렉션 및 드롭에 대한 관련 정보를 제출해야 합니다.

운영팀 담당자가 제공한 BOX 폴더 내 엑셀 템플릿(정보/데이터)을 작성해 주십시오. 작성 완료 후, 첨부된 메타 파일과 이미지 파일을 동일한 위치에 업로드하십시오.
{% endstep %}

{% step %}
## NFT 컬렉션 등록

운영 지원<a href="#id-2.-register-nft-collections" id="id-2.-register-nft-collections"></a>

채널에 제출하신 정보는 Unifi Apps 데이터베이스에 등록됩니다. 이 단계는 Unifi Apps 팀이 귀하를 대신하여 수행할 수 있습니다.

(앱은 NFT 컬렉션의 여러 계약에 할당될 수 있습니다.)
{% endstep %}

{% step %}
## 앱이 할당된 NFT 컬렉션 계약<a href="#id-3.-deploy-contract-for-nft-collections-assigned-mini-dapp" id="id-3.-deploy-contract-for-nft-collections-assigned-mini-dapp"></a>

배포

NFT 컬렉션 계약은 Unifi App 서버에서 배포됩니다. 계약이 확인되면 앱을 할당할 수 있습니다(앱에 계약 권한을 부여하는 것이 아니라, 유니파이 앱스 내에서 앱의 고유 계약을 매칭하는 단계입니다).
{% endstep %}

{% step %}
## 드롭 설정<a href="#id-4.-set-drops" id="id-4.-set-drops"></a>


유니파이 앱스 팀이 1단계에서 수집한 정보로 드롭을 설정합니다. 드롭 설정 시 NFT 발행 한도가 필요합니다. 구성된 드롭 내에는 상세 판매 계획을 등록하기 위해 여러 단계를 추가할 수 있습니다. 각 단계의 판매 일정이 중복될 수 없음을 유의하십시오.

드롭 내 구성된 단계는 유니파이 앱의 마켓 탭에 표시됩니다. 각 단계의 지정된 시간이 도래하면 사용자는 귀하의 NFT를 민팅할 수 있습니다.
{% endstep %}

{% step %}
## 구성 가능한 드롭

* ### 에어드랍 단계
  * 앱 내 특정 지갑이 에어드랍 목적으로 NFT를 수령할 수 있도록 하는 기능입니다. 에어드랍 단계에 대한 자세한 내용은 &#x27;NFT 발행 방법&#x27; 페이지를 참조하세요.
* ### 프리세일 단계
  * 사전 등록된 사용자(허용 목록)만 구매할 수 있는 단계입니다. 허용 목록 단계를 추가하려면 운영 지원 채널로 문의하세요.
* ### 공개 단계
  * Unfi 앱의 모든 사용자가 구매할 수 있는 단계입니다. 공개 단계는 드롭의 모든 단계 중 가장 마지막에 배치해야 하며, 공개 단계 시작 전까지 단계 추가가 가능합니다.
{% endstep %}

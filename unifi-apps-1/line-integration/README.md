# LINE

<mark style="color:red;">통합**Unifi Apps는 LIFF SDK를 사용하여 LINE 버전 웹 애플리케이션을 구축해야 합니다.****LINE 버전은 LINE 미니 앱 또는 LINE 로그인 LIFF 형태로 구축할 수 있습니다.**</mark>

## LINE 미니 앱

### 소개

공식 문서 : [LINE 미니 앱](https://developers.line.biz/en/reference/line-mini-app/)

LINE MINI 앱의 장점을 활용하여 LINE에서 사용자에게 최상의 서비스 경험을 제공하세요

### LINE MINI 앱이란?

<figure><img src="../../.gitbook/assets/LINEMINIApp_1.png" alt="" width="375"><figcaption></figcaption></figure>LINE 미니 앱은 사용자의 다양한 라이프스타일 요구를 충족하는 서비스를 LINE 내에서 제공할 수 있는 웹 애플리케이션입니다. 기업은 LINE 미니 앱의 장점을 활용해 사용자에게 편리한 서비스 경험을 제공함과 동시에 온라인과 오프라인 사용자 데이터에 접근할 수 있습니다.

### 간편한 서비스 활성화로 이탈 방지

<figure><img src="../../.gitbook/assets/LINEMINIApp_2.png" alt="" width="375"><figcaption></figcaption></figure>LINE MINI 앱은 LINE 내에서 바로 이용 가능하여 서비스 앱 다운로드나 복잡한 가입 절차가 필요 없습니다. 이는 서비스 이탈률을 최소화하고 전환율(CVR)을 향상시킵니다.

### LINE 내외부에서의 빠른 접근과 반복 이용

<figure><img src="../../.gitbook/assets/LINEMINIApp_3.png" alt="" width="563"><figcaption></figcaption></figure>LINE 미니앱은 거의 모든 곳에서 즉시 실행됩니다. LINE 홈 탭, 공식 계정, 웹 링크 및 QR 코드와 같은 외부 채널을 통한 접근이 지원됩니다. 이러한 마찰 없는 빠른 실행은 사용자 재사용 증가의 직접적인 요인입니다.

### 서비스 관련 정보 사용자 알림 용이성

<figure><img src="../../.gitbook/assets/LINEMINIApp_4.png" alt="" width="563"><figcaption></figcaption></figure>서비스 메시지를 통해 사용자가 공식 계정을 친구 추가하지 않았더라도 주문 확인 등 중요한 거래 관련 알림을 무료로 발송할 수 있습니다.

### LINE Talk을 통한 친구 공유 용이성

<figure><img src="../../.gitbook/assets/LINEMINIApp_5.png" alt="" width="563"><figcaption></figcaption></figure>사용자는 LINE 미니 앱에서 획득한 예약 정보나 쿠폰을 LINE Talk 친구들과 공유할 수 있어 서비스 홍보 효과가 증대됩니다.

## LINE 로그인 LIFF

### 소개

공식 문서 : [LINE 프론트엔드 프레임워크(LIFF)](https://developers.line.biz/en/docs/liff/overview/)

### LINE 사용자와 연결되는 웹 앱

LINE 프론트엔드 프레임워크(LIFF)는 LY Corporation이 제공하는 웹 애플리케이션 프레임워크입니다. 웹 앱에 LIFF SDK를 통합하면 LINE 플랫폼이 제공하는 정보에 접근하거나 LINE 앱의 기능을 활용할 수 있습니다.

### LINE 로그인 및 LINE 플랫폼 연동

<figure><img src="../../.gitbook/assets/p2-dark.svg" alt=""><figcaption></figcaption></figure>LIFF는 LINE 로그인과 연동됩니다. 복잡한 설정 없이도 LINE 플랫폼의 인증 절차를 통해 사용자의 프로필 정보에 안전하게 접근할 수 있습니다.

### 공유 대상 선택기: 메시지 전송을 위한 강력한 도구

<figure><img src="../../.gitbook/assets/p3-dark.png" alt=""><figcaption></figcaption></figure>공유 대상 선택기는 LIFF 앱을 통해 선택할 수 있는 LINE 친구에게 메시지를 전송하는 도구입니다. 채팅방을 열지 않아도 개발자가 LINE 친구에게 메시지를 보낼 수 있으므로, 이 기능을 활용해 사용자와 서비스를 연결하거나 정보를 전송하며 소셜 요소를 추가할 수 있습니다.

### 크로스 브라우저 지원

<figure><img src="../../.gitbook/assets/p4-dark.png" alt=""><figcaption></figcaption></figure>LIFF는 크로스 브라우저 호환성을 지원합니다. 대부분의 API는 데스크톱 브라우저와 호환되므로 일반 웹 브라우저와 LINE 앱 모두에서 작동합니다.

### 유틸리티 제공

<figure><img src="../../.gitbook/assets/p5-dark.png" alt=""><figcaption></figcaption></figure>LIFF 개발자를 지원하기 위해 다양한 편리한 유틸리티를 제공합니다. LINE 앱 버전이나 기기 OS 유형 등의 정보를 쉽게 조회할 수 있습니다. 또한 기본적으로 QR 코드 리더 기능이 제공됩니다.
